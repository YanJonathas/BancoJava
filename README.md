# Sistema Bancário Java

Simulação de um sistema bancário desenvolvido em Java com foco em 
Programação Orientada a Objetos.

## Como rodar
1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute o arquivo `Main.java`

## Funcionalidades
- Abertura de conta personalizada
- Saque com cheque especial e taxa de 20%
- Depósito
- Pagamento de boleto
- Extrato com histórico de transações
- Validação de entradas com try/catch

## Conceitos aplicados
- Encapsulamento
- Construtores
- ArrayList
- Switch/Case
- Try/Catch
